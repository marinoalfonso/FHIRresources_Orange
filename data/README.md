# Data

## Source

This project uses **synthetic patient records** generated by [Synthea™](https://synthea.mitre.org/), an open-source patient population simulator developed by MITRE Corporation. Synthea produces realistic but entirely fictional medical records — no real patient data is involved.

The records are provided in **FHIR R4 JSON format**, where each file represents a single patient's complete medical history as a FHIR Bundle.

---

## Downloading the Data

Pre-generated synthetic datasets are available for free at:

👉 [https://synthea.mitre.org/downloads](https://synthea.mitre.org/downloads)

Recommended download for this project: **"10,000 Sample Synthetic Patient Records, FHIR R4"**

After downloading, extract the archive and point the `fhirCaricamento.py` widget to the folder containing the `.json` files.

---

## Why Data is Not Included

The Synthea JSON files are not committed to this repository because:

1. The full datasets are large (hundreds of MB).
2. They are freely and publicly available from the official Synthea website.
3. Users may prefer to generate their own custom datasets using the [Synthea CLI](https://github.com/synthetichealth/synthea).

---

## Generating Custom Data (Optional)

If you want to generate your own synthetic population:

```bash
# Clone and build Synthea
git clone https://github.com/synthetichealth/synthea.git
cd synthea
./run_synthea -p 1000        # generate 1000 patients
```

Output files will appear in `synthea/output/fhir/`. Use those JSON files as input for the Orange widgets.
